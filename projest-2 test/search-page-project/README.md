# Search Page Project

This project implements a simple search page with two search bars, styled according to a specified mockup. The search bars are designed to stretch the full width of the page, with appropriate margins and spacing.

## Project Structure

```
search-page-project
├── src
│   ├── images          # Directory containing images used in the search page
│   ├── fonts           # Directory containing font files used in the project
│   ├── styles
│   │   └── main.css    # CSS styles for the search page
│   └── index.html      # HTML structure for the search page
└── README.md           # Documentation for the project
```

## Files Overview

- **src/index.html**: Contains the HTML structure for the search page, including two search bars with specific attributes and styling.
  
- **src/styles/main.css**: Contains the CSS styles for the search page, implementing the design mockup, including hover and focus states for the search bars, and custom font integration.

- **src/images**: Contains images used in the search page.

- **src/fonts**: Contains font files used in the project.

## Setup Instructions

1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Open `src/index.html` in a web browser to view the search page.
4. Ensure that the `src/styles/main.css` file is linked correctly in the HTML for styles to be applied.

## Usage

The search page is designed to be user-friendly, with two distinct search bars for different functionalities. The large search bar includes an input field that captures keypress events, while both search bars are styled to provide a consistent user experience.
